# 🛒 Marketplace Frontend

Frontend del proyecto **Marketplace**, desarrollado con **React + Vite** como Trabajo Práctico Integrador.

---

# 🚀 Tecnologías utilizadas

- React
- Vite
- Redux Toolkit
- React Router DOM
- Axios
- JavaScript
- CSS

---

# 📋 Funcionalidades

## 👤 Usuario

- Registro de usuario
- Inicio de sesión
- Catálogo de productos
- Búsqueda de productos
- Filtro por marca
- Detalle de producto
- Selección de talle
- Carrito de compras
- Favoritos
- Checkout
- Perfil de usuario

---

## 🛍️ Administrador

- Dashboard administrativo
- Gestión de productos
- Crear productos
- Editar productos
- Eliminar productos
- Carga de imágenes
- Gestión de marcas
- Gestión de categorías

---

# 💰 Gestión de productos

Cada producto permite:

- Marca
- Categoría
- Imagen
- Precio
- Descuento
- Variantes por talle
- Stock

# ⚙️ Instalación

Clonar el repositorio:

```bash
git clone https://github.com/Manoblia/marketplace-frontend.git
```

Instalar dependencias:

```bash
npm install
```

Ejecutar el proyecto:

```bash
npm run dev
```

Compilar para producción:

```bash
npm run build
```
